# Get Started Go Backend

Dự án Go backend sử dụng Gin framework.

## Cấu trúc thư mục

- `cmd/` - Entry points
- `internal/` - Code riêng project
- `pkg/` - Reusable packages
- `configs/` - Configuration files
- `migrations/` - Database migrations
- `tests/` - Test files

## Chạy ứng dụng

\`\`\`bash
go run cmd/server/main.go
\`\`\`

## API Endpoints

- GET `/hello` - Simple hello endpoint
